<h1><strong>Job Insights</strong></h1>

<h2><strong>Descrição</strong></h2>
<p align="justify">
  Neste projeto, são implementadas análises a partir de um conjunto de dados sobre empregos, com dados extraídos do Glassdoor e disponibilizados via Kaggle.
  A proposta é construir funções de leitura, filtragem e validação de regras sobre vagas, praticando fundamentos sólidos de Python e garantindo qualidade com testes automatizados.
</p>

<h2><strong>Funcionalidades</strong></h2>
<ul>
  <li align="justify">Leitura de arquivos CSV contendo dados de vagas de emprego.</li>
  <li align="justify">Implementação de funções para filtrar vagas por tipo, indústria, faixa salarial, entre outros critérios.</li>
  <li align="justify">Validação de regras de negócio, como consistência de dados e formatos esperados.</li>
  <li align="justify">Cobertura de testes automatizados utilizando Pytest para garantir a robustez das funções implementadas.</li>
</ul>

  <h1><strong>Job Insights</strong></h1>

  <h2><strong>Description</strong></h2>
  <p align="justify">
    In this project, analyses are implemented from a job dataset, originally extracted from Glassdoor and made available via Kaggle.
    The goal is to build file reading, filtering and validation utilities, practicing Python fundamentals and ensuring quality through automated tests.
  </p>

  <h2><strong>Features</strong></h2>
  <ul>
    <li align="justify">Reading CSV files containing job listing data.</li>
    <li align="justify">Implementing functions to filter job listings by type, industry, salary range, among other criteria.</li>
    <li align="justify">Validating business rules, such as data consistency and expected formats.</li>
    <li align="justify">Automated test coverage using Pytest to ensure robustness of implemented functions.</li>
  </ul>
